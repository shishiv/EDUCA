# Wireframe: Tela de Login (Tablet 768×1024)

```
+--------------------------------------------------------------------+
|                        [Logo] [Nome da Aplicação]                  |
+--------------------------------------------------------------------+
|                                                                    |
|                                                                    |
|                +----------------------------------+                |
|                | [Campo: E-mail @fronteira...]    |                |
|                +----------------------------------+                |
|                +----------------------------------+                |
|                | [Campo: Senha]                   |                |
|                +----------------------------------+                |
|                                                                    |
|                [Botão: Entrar]                                     |
|                [Botão: Entrar com Google] (opcional)               |
|                [Botão: Entrar com Microsoft] (opcional)            |
|                [Botão: Entrar com GOV] (opcional)                  |
|                                                                    |
|                [Link: Esqueci minha senha]                         |
|                [Link: Criar conta] (visível para perfis autorizados)|
|                                                                    |
|                [Feedback de erro: Usuário ou senha incorretos]     |
|                                                                    |
+--------------------------------------------------------------------+
|                [Footer minimalista: © Prefeitura de Fronteira]     |
+--------------------------------------------------------------------+
```

---

## Legendas e Observações

- **[Logo]**: Placeholder para logomarca da prefeitura, alinhado à esquerda ou centralizado.
- **[Nome da Aplicação]**: Exemplo: "Gestão Educacional", ao lado ou abaixo do logo.
- **Campos**: Inputs centralizados, largura máxima 400px.
- **Botões**: “Entrar” em azul (#1D4ED8), demais opcionais em tons de cinza.
- **Links**: Texto menor, abaixo dos botões.
- **Feedback de erro**: Texto em vermelho, aparece abaixo dos campos.
- **Footer**: Discreto, centralizado.
- **Responsividade**: Campos e botões centralizados verticalmente na tela.
- **Tipografia**: Fonte sem serifa (ex: Inter, Montserrat).
- **Acessibilidade**: Labels associados aos inputs, área tocável ≥ 44×44 px.

---

## Placeholders

- Texto de exemplo: “Lorem ipsum”.
- Ícones genéricos para botões sociais.

---

## Feedback de Estado

- Não exibido nesta tela, mas login pode mostrar toast/snackbar para sucesso/erro de autenticação.
