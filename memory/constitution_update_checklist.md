# Constitution Update Checklist

Use this checklist when updating the constitution to ensure all stakeholders and documentation remain aligned.

## Pre-Update Review
- [ ] Identify specific constitutional principles that need changes
- [ ] Document rationale for each proposed change
- [ ] Review impact on existing specifications and implementations
- [ ] Consult with educational domain experts if needed

## Educational Domain Validation
- [ ] Changes align with Brazilian educational regulations
- [ ] Attendance tracking requirements remain compliant
- [ ] Student data requirements meet municipal needs
- [ ] Multi-school architecture principles maintained

## Technical Validation
- [ ] Technology stack choices remain consistent
- [ ] Security principles (RLS, JWT) are preserved
- [ ] Performance targets are realistic and measurable
- [ ] Integration with existing gestao_fronteira codebase considered

## Documentation Updates Required
- [ ] Update CLAUDE.md with new constitutional requirements
- [ ] Modify spec templates to reflect new principles
- [ ] Update plan templates with new quality gates
- [ ] Revise task templates to include new standards

## Implementation Impact Assessment
- [ ] Existing code compliance with new constitution
- [ ] Required refactoring identified and estimated
- [ ] Migration path for non-compliant components planned
- [ ] Testing strategy updated for new requirements

## Stakeholder Communication
- [ ] Changes communicated to development team
- [ ] Educational stakeholders informed of compliance changes
- [ ] Project timeline adjusted if significant changes required
- [ ] Risk assessment updated based on constitutional changes

## Validation Steps
- [ ] Constitution reviewed by senior developer
- [ ] Educational compliance verified by domain expert
- [ ] Technical feasibility confirmed
- [ ] Timeline impact assessed and communicated

## Post-Update Actions
- [ ] All project documentation updated
- [ ] Development team briefed on changes
- [ ] Quality gates updated in CI/CD pipeline
- [ ] Next specification created with new constitutional requirements
