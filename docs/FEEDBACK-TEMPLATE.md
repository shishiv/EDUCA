## Como criar o formulario no Google Forms

1. Acesse [forms.google.com](https://forms.google.com)
2. Clique em **"Em branco"** para criar novo formulario
3. Adicione o titulo e descricao conforme acima
4. Para cada pergunta:
   - Clique no **"+"** para adicionar pergunta
   - Digite o texto da pergunta
   - Selecione o tipo (multipla escolha, escala, paragrafo)
   - Adicione as opcoes se aplicavel
5. Em **Configuracoes** (icone de engrenagem):
   - Desmarque "Coletar enderecos de email"
   - Desmarque "Limitar a 1 resposta"
6. Clique em **"Enviar"** para obter o link de compartilhamento
7. Compartilhe o link no grupo WhatsApp piloto
