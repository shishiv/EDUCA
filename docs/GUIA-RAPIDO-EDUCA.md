# EDUCA - Guia Rapido de Uso

**Sistema de Gestao Escolar - Prefeitura de Fronteira**

**Versao:** Piloto v2.1

---

## 1. Acesso ao Sistema

### Como acessar

1. Abra o navegador (Chrome, Firefox ou Edge)
2. Digite o endereco: **[URL_PRODUCAO]**
3. Aguarde a pagina carregar

### Fazendo login

1. Digite seu **email** no campo "Email"
2. Digite sua **senha** no campo "Senha"
3. Clique no botao **"Entrar"**

### Primeiro acesso

- Se voce ainda nao tem credenciais, solicite ao administrador do sistema
- Contato: **[CONTATO_ADMIN]**

---

## 2. Navegacao Basica

### Menu lateral (Sidebar)

O menu fica do lado esquerdo da tela. Clique nos icones para acessar:

| Icone | Funcao |
|-------|--------|
| Casa | Dashboard (pagina inicial) |
| Pessoas | Alunos |
| Grupos | Turmas |
| Lista | Chamada (frequencia) |
| Livro | Diario de Classe |
| Grafico | Relatorios |

### Seletor de escola (apenas administradores)

- Se voce administra mais de uma escola, selecione qual deseja visualizar
- O seletor aparece no topo da pagina
- A escola selecionada afeta todos os dados exibidos

### Perfis de usuario

| Perfil | O que pode fazer |
|--------|------------------|
| **Professor** | Registrar frequencia e atividades da sua turma |
| **Diretor** | Visualizar dados de toda a escola |
| **Secretario** | Gerenciar alunos, turmas e matriculas |

## 3. Tarefas Diarias - Professor

### Registrar frequencia (Chamada)

**Passo 1:** Clique em **"Chamada"** no menu lateral

**Passo 2:** Selecione a **turma** desejada

**Passo 3:** Verifique a **data** (automaticamente mostra hoje)

**Passo 4:** Para cada aluno:
- Clique em **P** = Presente
- Clique em **F** = Falta
- Clique em **J** = Falta Justificada

**Passo 5:** Clique em **"Salvar"** para confirmar

**Importante:**
- O horario limite para registrar frequencia e **18h** (horario de Brasilia)
- Apos esse horario, o sistema trava automaticamente
- Em caso de problemas, contate o suporte imediatamente

### Registrar atividades (Diario de Classe)

**Passo 1:** Clique em **"Diario"** no menu lateral

**Passo 2:** Selecione a **turma**

**Passo 3:** Clique em **"Nova Atividade"**

**Passo 4:** Preencha os campos:
- **Titulo** da atividade
- **Descricao** do que foi trabalhado
- **Campos de Experiencia BNCC** (selecione os que se aplicam)

**Passo 5:** Escolha uma opcao:
- **"Salvar Rascunho"** - salva mas permite editar depois
- **"Finalizar"** - salva definitivamente (nao pode mais editar)

## 4. Tarefas Diarias - Diretor/Secretario

### Visualizar Dashboard

O Dashboard e a pagina inicial. Mostra:

- **Total de alunos** matriculados
- **Total de turmas** ativas
- **Taxa de frequencia** do dia/semana/mes
- **Alertas importantes** (se houver)

### Consultar alunos

**Passo 1:** Clique em **"Alunos"** no menu lateral

**Passo 2:** Use o campo de busca para encontrar por nome

**Passo 3:** Clique no nome do aluno para ver detalhes

### Consultar turmas

**Passo 1:** Clique em **"Turmas"** no menu lateral

**Passo 2:** Veja a lista de turmas com:
- Nome da turma
- Professor responsavel
- Quantidade de alunos

### Acompanhar frequencia geral

**Passo 1:** Clique em **"Relatorios"** no menu lateral

**Passo 2:** Selecione **"Frequencia"**

**Passo 3:** Escolha o periodo desejado

### Alertas Bolsa Familia

O sistema exibe alertas automaticos quando:

- Um aluno do Bolsa Familia tem frequencia **abaixo de 80%**
- O alerta aparece em destaque no Dashboard
- **Acao necessaria:** Entrar em contato com a familia
