04

Uso do Logo

Para manter a consistência da marca, siga estas diretrizes de aplicação do logo.

✓
Fundos claros neutros

Use a versão principal com gradiente em fundos brancos ou tons claros.

✓
Fundos escuros

Use a versão com cores mais claras para garantir contraste.

✕
Não distorça

Nunca estique ou comprima o logo. Mantenha as proporções.

✕
Não altere as cores

Use apenas as cores definidas neste manual.

✕
Evite fundos vibrantes

Não use em fundos coloridos que competem com o logo.

✕
Não reduza opacidade

O logo deve sempre estar em 100% de opacidade.

06

Co-branding

Diretrizes para uso conjunto do logo EDUCA com a identidade da Prefeitura de Fronteira.

Logo EDUCA + Prefeitura

Em documentos oficiais e materiais institucionais, os logos devem aparecer com separação clara e tamanhos proporcionais.

PREFEITURA DE
FRONTEIRA

Quando usar juntos

  • ✓ Documentos oficiais (boletins, relatórios)
  • ✓ Materiais de lançamento e divulgação
  • ✓ Apresentações institucionais
  • ✓ Cartazes e banners para escolas

Quando usar EDUCA sozinho

  • ✓ Interface do sistema (login, menus)
  • ✓ Ícone do app e favicon
  • ✓ Manual do usuário digital
  • ✓ Comunicações internas do sistema
