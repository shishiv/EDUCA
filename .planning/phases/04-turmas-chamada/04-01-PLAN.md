Create TurmaCard component with:

1. **Props interface:**
   ```typescript
   interface TurmaCardProps {
     turma: {
       id: string
       nome: string
       serie: string
       turno: 'matutino' | 'vespertino' | 'integral'
       escola: { nome: string }
       alunos_matriculados: number
       capacidade: number
       professor?: { nome: string } | null
       ativo: boolean
     }
     onChamada?: (turmaId: string) => void
     onDiario?: (turmaId: string) => void
   }
   ```

2. **Serie color mapping function** (getSerieColor):
   - Infantil (bercario, maternal, pre): `pink`
   - Fundamental I (1-5 ano): `orange`
   - Fundamental II (6-9 ano): `violet`
   - Fallback: `gray`

3. **Card structure:**
   - Top color band: `h-2 rounded-t-lg` with gradient from serie color
   - Use dynamic Tailwind classes via safelist or inline style for gradient colors
   - CardHeader: turma.nome (title), escola.nome (description)
   - CardContent: stats row (alunos/capacidade, turno badge), action buttons row
   - Status badge if turma.ativo is false

4. **Hover effects:**
   - `hover:shadow-md hover:-translate-y-0.5 transition-all duration-200`
   - `cursor-pointer` on card (for click navigation)

5. **Action buttons:**
   - "Fazer Chamada" (primary, small) - calls onChamada, stops propagation
   - "Ver Diario" (outline, small) - calls onDiario, stops propagation
   - Both use stopPropagation to prevent card click

6. **Card click handler:**
   - Wrap in Link to `/dashboard/turmas/${turma.id}` OR use onClick with router.push
   - Ensure buttons don't trigger navigation
