## Decisions Made

| Decision | Rationale |
|----------|-----------|
| Keep API service class exports | Even when only singleton instances are used, class exports enable type inference |
| Remove entire directories when all files unused | Cleaner than individual file removal |

| Delete validation modules entirely | attendance.ts and performance.ts had zero consumers |
| Keep lib/services/attendance-immutability.ts | Used by lib/api/attendance.ts which IS used in production |
