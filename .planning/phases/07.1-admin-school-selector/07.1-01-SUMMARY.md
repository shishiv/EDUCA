## Decisions Made
- **SessionStorage over localStorage:** Per CONTEXT.md, selection persists only in session (clears on browser close)
- **Hydration-safe pattern:** Read sessionStorage in useEffect to avoid SSR/CSR mismatch
- **Auto-select for single-school users:** diretor/secretario/professor with escola_id get it auto-selected
- **shouldShowSelector rules:**
  - admin, gestor_sme: always true
  - diretor, secretario: always false
  - coordenador, professor: true only if escolas.length > 1 (future multi-escola support)
  - responsavel: always false
