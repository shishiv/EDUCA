  <action>
Create EscolaContext following the SearchContext pattern exactly:

1. **Types:**
```typescript
interface Escola {
  id: string
  nome: string
}

interface EscolaContextType {
  // Data
  escolas: Escola[]
  selectedEscolaId: string | null
  selectedEscola: Escola | null

  // Loading
  loading: boolean

  // Actions
  selectEscola: (id: string | null) => void
  clearSelection: () => void

  // Computed
  shouldShowSelector: boolean
}
```

2. **Provider implementation:**
- Use `useAuth()` to get userProfile (tipo_usuario, escola_id)
- Fetch escolas from Supabase on mount (only when userProfile is available to avoid race condition)
- For admin/gestor_sme: fetch all escolas
- For other roles: use userProfile.escola_id (single escola, don't fetch)
- Compute `shouldShowSelector`:
  - admin, gestor_sme: always true
  - diretor, secretario: always false
  - coordenador, professor: true if escolas.length > 1 (future: currently single escola)

3. **SessionStorage persistence (avoid hydration mismatch):**
- Key: `gestao-fronteira-selected-escola`
- Read sessionStorage in useEffect only (not useState initial value)
- Write to sessionStorage on selection change
- Clear on clearSelection()

4. **selectedEscola derived:**
```typescript
const selectedEscola = useMemo(() =>
  escolas.find(e => e.id === selectedEscolaId) || null,
  [escolas, selectedEscolaId]
)
```

5. **Export:**
- EscolaProvider component
- useEscola hook (throws if used outside provider)
