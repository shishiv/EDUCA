# Phase 7.1: Admin School Selector - Context

**Gathered:** 2026-01-19
**Status:** Ready for planning

<domain>
## Phase Boundary

Seletor de escola persistente para perfis multi-escola conseguirem acessar dados escola-scoped (turmas, alunos, frequência). O seletor aparece para perfis que podem ver múltiplas escolas; perfis com escola única não vêem o componente.

</domain>

<decisions>
## Implementation Decisions

### Posicionamento do seletor
- Header mostra escola atual (nome completo), sidebar permite trocar
- Se nenhuma escola selecionada: placeholder claro "Selecione uma escola" com destaque visual
- Visibilidade (contextual vs sempre): Claude decide

### Experiência de seleção
- Combobox com busca (funciona bem para 9 escolas)
- Nenhuma escola selecionada por padrão — força seleção explícita
- Comportamento ao trocar escola: Claude decide

### Regras de perfil e escopo
| Perfil | Escolas visíveis | Seletor |
|--------|------------------|---------|
| Administrador | Todas (9) | Sempre visível |
| Gestor SME | Todas (9) | Sempre visível |
| Secretário de Escola | Só sua escola | Escondido |
| Diretor | Só sua escola | Escondido |
| Coordenador | Escolas onde trabalha | Visível se >1 |
| Professor | Escolas onde trabalha | Visível se >1 |

- Se usuário só tem UMA escola: esconder seletor (não mostrar bloqueado)

### Persistência
- Seleção persiste apenas na sessão (limpa ao logout)
- Sincronização entre abas: Claude decide

### Integração com páginas
- Páginas escola-scoped sem escola selecionada: mostrar vazio + botão "Selecionar escola"
- Claude identifica quais páginas precisam de escola_id no código
- URL com ou sem escola_id: Claude decide
- Dashboard admin: mostra dados agregados de todas + destaque da escola selecionada

### Claude's Discretion
- Visibilidade do seletor (contextual vs sempre visível)
- Comportamento ao trocar escola (recarregar dados vs redirecionar)
- Sincronização entre abas
- Estrutura de URL (/turmas vs /escolas/:id/turmas)
- Identificar quais páginas precisam de escola_id

</decisions>

<specifics>
## Specific Ideas

- 9 escolas no sistema de Fronteira — lista simples com busca é suficiente
- Coordenadores são professores que trabalham em várias escolas e precisam ver suas turmas em cada uma
- Gestor da Secretaria Municipal é perfil separado do Secretário de Escola
