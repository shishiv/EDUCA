<task type="auto">
  <name>Task 1: Create GitHub Actions CI workflow</name>
  <files>.github/workflows/ci.yml</files>
  <action>
    Create `.github/workflows/ci.yml` with the following configuration:

    ```yaml
    name: CI

    on:
      push:
        branches: [main]
      pull_request:
        branches: [main]

    jobs:
      quality:
        name: Type Check and Lint
        runs-on: ubuntu-latest
        defaults:
          run:
            working-directory: gestao_fronteira

        steps:
          - name: Checkout
            uses: actions/checkout@v4

          - name: Setup pnpm
            uses: pnpm/action-setup@v4
            with:
              version: 9

          - name: Setup Node.js
            uses: actions/setup-node@v4
            with:
              node-version: '20'
              cache: 'pnpm'
              cache-dependency-path: gestao_fronteira/pnpm-lock.yaml

          - name: Install dependencies
            run: pnpm install --frozen-lockfile

          - name: Type check
            run: pnpm typecheck

          - name: Lint
            run: pnpm lint
    ```

    Key points:
    - Uses pnpm 9 (matches local setup)
    - Node 20 (LTS)
    - Caches pnpm store for faster builds
    - Runs from gestao_fronteira/ directory
    - Runs on both PR and push to main
  </action>
  <verify>

    - Contains required steps: typecheck and lint
  </verify>
