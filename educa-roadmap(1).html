Pontos de Atenção

  • LGPD: Definir DPO (encarregado de dados), documentar bases legais para tratamento de dados de menores.
  • Backup: Definir política de retenção, local de armazenamento (cloud provider), processo de restore.
  • WhatsApp: Utilizar API oficial (Evolution API já configurada) para evitar bloqueios.
  • Educacenso: Validar layout do arquivo de exportação com a Secretaria antes de implementar.
  • Treinamento: Planejar capacitação presencial para professores e gestores antes do lançamento.
  • Piloto: Sugestão de rodar piloto em 1-2 escolas antes do rollout completo.
