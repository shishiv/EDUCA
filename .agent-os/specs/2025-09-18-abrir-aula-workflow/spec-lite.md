# Abrir Aula Workflow - Lite Summary

Implement the workflow for teachers to formally open class sessions before marking attendance, ensuring immutable attendance records for Brazilian educational compliance through a mobile-optimized interface with time-locked submissions.

## Key Points
- Teachers must open a class session ("Abrir aula") before marking any attendance
- Once attendance is saved, records become permanently locked to prevent retroactive changes
- Mobile-first design optimized for teacher tablets and smartphones with fast attendance marking
